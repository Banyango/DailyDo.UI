# DailyDo.UI

This is the frontend SPA for DailyDo 

To run install dev dependencies

```npm i --no-optional```

Then run 

``npm start``

This will run the UI application on ``localhost:3000``

The App landing page is /app.

See *DailyDo.Server* for running the backend.
